Этот файл содержит описания скриншотов для документации.

## Как сделать скриншоты:

1. Запустите `./ai-bot.exe --config` и настройте бота
2. Запустите `./ai-bot.exe` 
3. Откройте браузер и перейдите на нужные страницы
4. Сделайте скриншоты в хорошем качестве (1920x1080 или выше)
5. Сохраните в папку `img/` с соответствующими именами
6. Оптимизируйте размер файлов (рекомендуется PNG или WebP)

## Рекомендации по скриншотам:

- Используйте светлую тему браузера для лучшей читаемости
- Покажите интерактивные элементы (открытые меню, hover эффекты)
- Включите несколько сообщений в чате для демонстрации диалога
- Убедитесь что текст читаемый и интерфейс понятный
- Покажите уникальные особенности каждой демо страницы
